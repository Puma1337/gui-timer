# gui-timer
GUI Timer for Coffee Brewing


## What's it do?
Displays a GTK percentage timer for tracking that cup of coffee you are brewing!

## Features
* A timer!
* Bean pouring sounds!
* A gong to let you know your brew is fininshed!
